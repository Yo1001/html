  15. What is HTML? How are HTML tags formed? Explain HTML comments?

      16. HTML, or HyperText Markup Language, is a standard markup language for creating web pages. It's a text-based approach to describing how content is structured within an HTML file. HTML tells a web browser how to display text, images, and other forms of multimedia on a webpage.

      HTML elements are the building blocks of a web page. They label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc..

      HTML tags are special keywords that specify how a web browser must format and display the content. Tags are wrapped in brackets < and >. They start with an open angle bracket (<) and close with a closed angle bracket (>). The ending tag has a forward slash before the name of the element.

      HTML comments can be used to explain the intended functionality of a section of code for another developer or your future self. Comments can also be used to assign someone a task or point out an error.

      Multi-line comments are those that span multiple lines. They begin with the start tag .
